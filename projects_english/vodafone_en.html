Experience at Vodafone Global Group (2010–2014)

Role: Senior Agile Tester, QA Technical Lead, Software Tester

Company: Vodafone Global Group (via Reply GmbH, Düsseldorf, Germany)

Overview

Over several years, I worked in various QA and technical leadership roles, contributing to large-scale international mobile projects. My responsibilities spanned quality assurance, test automation, mobile testing, project coordination, and tool development.

Main Responsibilities

  • Managed QA processes for up to 13 Vodafone mobile services and coordinated 9 cross-functional teams.
  • Planned and executed functional, regression, and performance testing on iOS, Android, Windows, and BlackBerry devices.
  • Developed automated test scripts using Selenium WebDriver and Java in Eclipse.
  • Coordinated ticketing and SLA tracking for service requests and change management.
  • Led Vodafone Wallet service integration (NFC-based payment solution with Visa).
  • Maintained documentation and device repositories via SharePoint and internal Wikis.
  • Created XML config files for MMS services and managed Vodafone Live portal styling via CSS.
  • Developed "Devicepedia", an Android app to display system data like RAM, CPU, and codec support.
  • Trained and onboarded new QA team members.
