Senior Agile Tester bei Qiagen (03.2015 – 02.2016)

Ort: Düsseldorf, Deutschland

Arbeitgeber: Brunel GmbH

Überblick

Verantwortlich für das agile Testing in der R&D-Softwareabteilung im Bereich molekulare Analyse- und Diagnosetechnologien. Mitarbeit in einem Scrumteam zur Entwicklung und Verifikation automatisierter Systeme.

Hauptverantwortlichkeiten

  • Design-Spezifikationen analysieren und bewerten.
  • Erstellung und Überprüfung von Testfällen in enger Zusammenarbeit mit Entwicklern.
  • Testplanung und Koordination von Tickets.
  • Technische Dokumentationserstellung.
  • Durchführung von Functional, Regression, UI und Exploratory Testing.
  • Datenbanktests (SQL-basiert) und Localization Testing.
  • Systemtests inkl. Log-Analyse und Testumgebungsaufbau.
  • Arbeiten mit Continuous Integration Prozessen.

Verwendete Technologien

Bamboo Polarion SVN JIRA XML SQL Server SQL Server Management Studio Visual Studio Confluence Notepad++ FastStone Capture Microsoft Office Suite

Branche

Medizintechnik / Bioinformatik
