Test Manager und Senior Agile Tester bei Arvato Systems (05.2016 – 05.2017)

Ort: Köln, Deutschland

Arbeitgeber: Ferchau Engineering GmbH

Überblick

Verantwortlich für agiles Testing in der Abteilung Digital Media Management zur Entwicklung und Verifikation des Video Production Management Systems.

Hauptverantwortlichkeiten

  • Beurteilung von Design-Spezifikationen.
  • Erstellung und Ausführung von Testfällen in verschiedenen Entwicklungsphasen (Scrum).
  • Testplanung, technische Dokumentation und Testfallkoordination.
  • Durchführung von API Testing, Functional Testing des Transcoding-Systems, Regression Testing, UI Testing und Exploratory Testing.
  • Datenbanktests und Loganalyse.
  • System- und Integrationstests sowie Vorbereitung der Testumgebung.
  • Einsatz von Continuous Integration und GIT-Versionierung.

Verwendete Technologien

  • Testing & CI: Microsoft Team Foundation Server, Microsoft Test Manager, TestComplete, SoapUI, Fiddler
  • Sprachen & Daten: SQL Server, SQL Server Management Studio, XML, GIT, Log4View
  • Tools & Umgebungen: Visual Studio, VMware, Linux CentOS, MobaXterm, mRemoteNG, Microsoft Sharepoint, Notepad++, Microsoft Office Suite

Branche

Telekommunikation / Broadcast
